Personal Information Form Project

Overview:

This project is an easy to use website that uses Flask and lets users fill out a form with their personal data. A results page then shows the data that was submitted. For a better user experience, CSS is used to style the application.

Features:
- User-friendly form for collecting personal information including:
- Name
 -Age
 -Email
- Gender
- Date of Birth
- Phone
- Address
- Occupation
- Data validation to ensure all fields are filled out before submission.
- Display of submitted information on a separate results page.
Responsive design for accessibility on various devices.

Running the Application Locally

1. Clone the Repository:
- git clone <repository-url>
- cd <repository-directory>

2. Install Dependencies Make sure you have Flask installed. If you haven't installed it yet, run:
- pip install Flask

3. Run the Application In the terminal, navigate to the directory containing app.py and run:
- python app.py

4. Access the Application Open your web browser and go to http://127.0.0.1:5000/ to view the application. Now, you are good to go!
